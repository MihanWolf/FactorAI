# ItemData

**Тип:** Resource  
**Файл:** `Scripts/Item/item_data.gd`  
**Наследует:** Resource

## Описание
Чертёж предмета — описывает что такое "Радио" или "Тарелка" в общем виде. Один ресурс на весь тип предмета, не меняется в процессе игры. Хранится в `.tres` файле и регистрируется в [[ItemDatabase]].

Не путать с [[ItemInstance]] — это конкретное радио с царапинами и зелёными проводами, а `ItemData` это просто описание "что такое радио вообще".

## Зависимости
- [[ComponentSlotData]] — описание гнёзд для компонентов внутри предмета
- [[DisassembleEntry]] — что выпадает при разборке

## Переменные

| Имя                   | Тип                      | Описание                            |
| --------------------- | ------------------------ | ----------------------------------- |
| item_name             | String                   | название предмета                   |
| description           | String                   | описание                            |
| quality_levels        | Array[String]            | возможные уровни качества           |
| component_slots       | Array[ComponentSlotData] | гнёзда для компонентов              |
| tags                  | Array[String]            | теги для фильтрации и совместимости |
| sprite                | Texture2D                | иконка предмета                     |
| disassemble_materials | Array[DisassembleEntry]  | материалы при разборке              |
| item_type             | ItemType                 | тип предмета из enum                |

## Enum ItemType

| Значение   | Описание                      |
| ---------- | ----------------------------- |
| MATERIAL   | сырьё, материал               |
| ITEM       | обычный предмет               |
| CONSUMABLE | расходник                     |
| EQUIP      | экипировка                    |
| TOOL       | инструмент для стола разборки |
| ANY        | любой предмет                 |
