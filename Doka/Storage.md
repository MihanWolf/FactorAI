# Storage

**Тип:** Resource  
**Файл:** `Scripts/storage.gd`  
**Наследует:** Resource

## Описание
Универсальное хранилище предметов. Используется везде где нужно хранить [[ItemInstance]] — инвентарь игрока, телега, ящик, стол разборки. Хранит массив слотов [[InventorySlot]] и умеет добавлять/удалять предметы.

Будучи `Resource` а не `Node` — не привязан к сцене и может существовать просто как данные.

## Зависимости
- [[InventorySlot]] — слоты внутри хранилища
- [[ItemInstance]] — предметы которые хранятся

## Сигналы

| Сигнал | Параметры | Когда испускается |
|--------|-----------|-------------------|
| slot_changed | slot_index: int | содержимое слота изменилось (добавлен или убран предмет) |

## Переменные

| Имя | Тип | Описание |
|-----|-----|----------|
| storage_name | String | название хранилища, отображается в UI |
| slots | Array[InventorySlot] | слоты хранилища |

## Методы

### add_item(item) -> bool
Пытается положить предмет в первый подходящий свободный слот. Перебирает слоты по порядку и вызывает `slot.place_item()` на каждом. При успехе испускает `slot_changed`. Возвращает `false` если подходящего слота нет.

---

### remove_item_from_slot(index) -> ItemInstance
Вынимает предмет из конкретного слота по индексу. Испускает `slot_changed` если предмет был там. Возвращает вынутый предмет или `null`.

---

### is_full() -> bool
Проверяет заполнено ли хранилище — перебирает все слоты и ищет хотя бы один пустой. Возвращает `true` только если все слоты заняты.
