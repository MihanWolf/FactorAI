# InventorySlot

**Тип:** Resource  
**Файл:** `Scripts/inventory_slot.gd`  
**Наследует:** Resource

## Описание
Один слот внутри [[Storage]]. Знает какой тип предмета принимает и хранит ссылку на лежащий там [[ItemInstance]]. Это самая маленькая единица хранения в игре.

## Зависимости
- [[ItemData]] — ссылается на его enum `ItemType`
- [[ItemInstance]] — хранит экземпляр предмета

## Переменные

| Имя | Тип | Описание |
|-----|-----|----------|
| accepted_type | ItemData.ItemType | какой тип предметов принимает этот слот |
| item | ItemInstance | предмет лежащий в слоте, `null` если пусто |

## Методы

### can_accept(incoming) -> bool
Проверяет подходит ли предмет для этого слота по типу. 

Текущая логика: если `accepted_type == ANY` — принимает любой предмет. Иначе проверяет точное совпадение типа.


---

### place_item(incoming) -> bool
Пытается положить предмет в слот. Проверяет два условия:
1. Слот свободен (`item == null`)
2. Предмет подходит по типу (`can_accept()`)

Возвращает `false` если хотя бы одно условие не выполнено.

---

### remove_item() -> ItemInstance
Вынимает предмет из слота — обнуляет `item` и возвращает то что было. Возвращает `null` если слот был пуст.
