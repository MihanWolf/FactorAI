# InventoryUI

**Тип:** Node  
**Файл:** `Scripts/inventory_ui.gd`  
**Наследует:** Control

## Описание
UI для отображения любого [[Storage]]. Универсален — одинаково показывает и инвентарь игрока, и телегу, и ящик. Создаёт по одному [[SlotUI]] на каждый слот и подписывается на изменения хранилища чтобы обновляться автоматически.

## Зависимости
- [[Storage]] — хранилище которое отображает
- [[SlotUI]] — визуал одного слота, инстанциируется динамически
- [[InventorySlot]] — передаётся в каждый SlotUI

## Методы

### init(storage)
Инициализирует UI под конкретное хранилище. Вызывается один раз после добавления ноды в сцену.

Как работает:
1. Ставит название хранилища в заголовок
2. Очищает старые слоты если были
3. Создаёт по одному `SlotUI` на каждый слот из `storage.slots`
4. Подписывается на сигнал `storage.slot_changed` — при изменении слота автоматически обновляет нужный `SlotUI`
