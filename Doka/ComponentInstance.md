# ComponentInstance

**Тип:** Resource  
**Файл:** `component_instance.gd`  
**Наследует:** Resource

## Описание
Конкретный экземпляр компонента — корпус с потёртостями, зелёным цветом и 70% состоянием. В отличие от [[ComponentData]] который один на всех, `ComponentInstance` уникален для каждого компонента в мире.

Хранится в [[ItemInstance]] в словаре `components` по ключу `slot_id`.

## Зависимости
- [[ComponentData]] — чертёж этого компонента

## Переменные

| Имя             | Тип           | Описание                                                                            |
| --------------- | ------------- | ----------------------------------------------------------------------------------- |
| data            | ComponentData | чертёж этого компонента                                                             |
| condition       | float         | состояние от 0.0 (сломан) до 1.0 (идеален)                                          |
| color_overrides | Dictionary    | перекраска регионов: { "region_name": Color }                                       |
| custom_label    | String        | кастомное название, например "Советский динамик" — задел на будущее                 |
| is_locked       | bool          | прикручен болтами — нужно открутить перед снятием — задел на будущее                |
| required_tool   | String        | ⚠️ перенести в [[ComponentSlotData]] — инструмент привязан к слоту, не к экземпляру |

## Методы

### create_random(from_data) -> ComponentInstance
**Статический метод** — создаёт новый экземпляр компонента со случайными параметрами. Вызывается из [[ItemInstance]].create_random() при заполнении слотов.

Как работает:
1. Создаёт новый `ComponentInstance` и привязывает чертёж `from_data`
2. Генерирует случайное состояние от 0.2 до 1.0 (совсем сломанные не спавнятся)
3. Для каждого региона из `from_data.color_regions` генерирует случайный цвет

---

### get_condition_label() -> String
Переводит числовое состояние `condition` в читаемую строку для UI.

| condition | Результат |
|-----------|-----------|
| >= 0.9 | "Идеальное" |
| >= 0.7 | "Хорошее" |
| >= 0.4 | "Поношенное" |
| < 0.4 | "Сломанное" |
